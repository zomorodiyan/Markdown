# Markdown
## used for
* readme files(Github, etc)
* forum & blog posts
* many static site generators
## can format
* headings
* blocks of code
* lists, images
* emphasis
* blockquotes
* links
* horizontal rules

I Use VSCode with "Auto-Open Markdown Preview" extension
